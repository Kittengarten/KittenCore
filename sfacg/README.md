# SF轻小说报更

## 简介

基于 [sfacgTrack](https://github.com/DawnNights/sfacgTrack) 开发的 [SF轻小说](https://book.sfacg.com) QQ 群报更机器人插件
（仅保留图文功能）

## 使用方法

> 小说 `书号` | `关键词`

可用参数获取小说信息

> 更新测试 `书号` | `关键词`

可用参数测试小说报更功能

> 更新预览 `书号` | `关键词`

可用参数预览小说更新内容

> 添加报更 `书号` | `关键词`

可用参数添加小说自动报更

> 取消报更 `书号` | `关键词`

可用参数取消小说自动报更

> 查询报更

可查询当前小说自动报更
